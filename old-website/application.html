Daily Scripture is a Node.js application that sends daily scriptures to anyone that enters their phone number. Once the number has been submitted, the user will reqularly receive daily scriptures in the morning and at night. The inspiration for this project came from a software engineer on Youtube that made a program that sends good morning text through an API.

This app was created with HTML, CSS, and components of bootstrap. It utilizes two APIs, the first API retrieves the bible scripture. The second API will send the scripture as an SMS to the numbers, Node,js is used to accomplish this. Through this app I wanted to demostate the use of API after having learned how to user one.
