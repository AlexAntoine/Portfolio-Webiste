Process

The Daily Scripture app was created using HTML, CSSS, bootstrap, and Node.js for the back-end. This app operates as a newsletter signup. New users enter their phone number and every morning at 6am evenning at 9pm they will receive a scripture via SMS. The numbers will be stored using an array and the scriptures will bet gathered through an API and delivered to each participant via a SMS API. I creadted this application after having learned how to use API with Node.js.
